# Laundry Mart Web Project

A simple, semantic, and user-friendly web interface for a laundry service provider. This project includes a service pricing table, a customer booking form, and a contact footer.

## Table of Contents

Features
Code Structure
How to Run
Service Information

### The project consists of the following HTML components:

<table>: Used for the price list.
<form>: Used for customer bookings with required validation.
<footer>: Contains the contact paragraph: "Contact us at: info@laundrymart.com".

#### How to Run

1. Copy the HTML and CSS code provided into a file named index.html.
2. Open the file in any modern web browser (Chrome, Firefox, Safari, etc.).
